# Roblox GUI — Professional UI Design Skill

Creates polished, game-ready Roblox GUIs through the Roblox Studio MCP.

## Features
- Style presets (cartoon, sleek, fantasy, scifi) or custom via style-config.md
- Auto-wiring: buttons automatically open their frames
- Auto-animation: open/close tweens, hover effects, click press on every button
- Mandatory screenshot checklist before showing user
- Responsive scaling (all Scale, never pixel Offset)
- Tab content switching (actually swaps content, not just color)
- Creative variety (layered frames, rotated backgrounds, different layouts)
- Complete LocalScript templates for every GUI pattern

## Requirements
- Claude Code with Roblox Studio MCP

## Installation
Copy roblox-gui/ to ~/.claude/skills/roblox-gui/

Then edit style-config.md to set your game's style, or use a preset in your prompt.

## Usage
- "Build a shop button and shop frame, cartoon style"
- "Create a pet inventory screen"
- "Make a settings menu with sliders"
- "Add a daily rewards popup"
