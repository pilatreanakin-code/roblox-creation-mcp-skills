# Roblox Game Planner — Full Game Design from One Prompt

Generates a complete game design folder from a single game idea. Every document is production-ready with real numbers, formulas, and cross-references.

## What It Generates
- Master Game Design Document (GDD) — full game vision
- GUI Design — every screen, element, color, animation
- Models & Assets — every 3D model, icon, effect needed
- Progression & Economy — XP curves, pricing formulas, drop rates
- Monetization — GamePasses, DevProducts, pricing strategy
- Map & World — every zone with theme, enemies, unlock requirements
- Update Roadmap — 6-month content plan
- Scripts folder — one architecture doc per game system with code

## Requirements
- Claude Code

## Installation
Copy roblox-game-planner/ to ~/.claude/skills/roblox-game-planner/

## Usage
"Plan a pet simulator game where players collect magical creatures in a fantasy world"
