# Models & Assets Template — Required Sections

## 1. Art Style Reference
- Overall style description
- Color palette
- Reference images or games
- Polygon budget per category (characters, pets, environment props)

## 2. Characters / Avatars
For each: name, description, visual style, animations needed

## 3. Pets / Companions
For each pet:
| Name | Rarity | Visual Description | Animations | Stats |
|------|--------|-------------------|------------|-------|

## 4. Items / Equipment
For each item: name, category, visual description, icon style

## 5. Environment Assets
Per zone: list of props (trees, rocks, buildings, portals, NPCs)

## 6. UI Icons
Every icon needed: name, purpose, size, style (flat/outlined/3D)

## 7. Particle Effects
List of VFX: hatching, level up, rebirth, purchase, damage numbers, zone portals

## 8. Asset Tracking Table
| Asset Name | Category | Description | Priority | Status |
|-----------|----------|-------------|----------|--------|
| Pet_Dragon | Pets | Red cartoon dragon, flying idle | High | ❌ |

## 9. Naming Convention
- Pets: `Pet_[Name]_[Rarity]`
- Items: `Item_[Name]`
- Environment: `Env_[Zone]_[PropName]`
- Icons: `Icon_[Name]_[Size]`
