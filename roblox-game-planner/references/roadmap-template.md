# Update Roadmap Template — Required Sections

## 1. Launch Content (Day 1)
- What's in the game at launch
- How many zones, pets, items, features
- What's the "endgame" at launch

## 2. Update Schedule
- How often? (weekly, bi-weekly, monthly)
- What each update type contains:
  - Small update: bug fixes, balance tweaks, new codes
  - Medium update: new items, new eggs, new zone section
  - Large update: new zone, new system, new event

## 3. Planned Updates (first 3-6 months)
For each update:
| Update | Timeline | Content | New Zone? | New Pets? | Event? |
|--------|----------|---------|-----------|-----------|--------|
| 1.1 | Week 2 | Bug fixes + 2 new pets + codes | No | 2 | No |
| 1.2 | Week 4 | Zone 3 + new egg + gamepass | Yes | 5 | No |
| 1.3 | Week 6 | Halloween event | No | 3 event pets | Yes |

## 4. Seasonal Events
- Halloween (October): theme, exclusive rewards, duration
- Christmas (December): theme, exclusive rewards, duration
- Easter (April): theme, exclusive rewards, duration
- Summer (July): theme, exclusive rewards, duration

## 5. Community Engagement
- Social media strategy (codes via Twitter, teasers, polls)
- Group rewards (join group → free item)
- Referral system (invite friend → both get reward)
- Code release schedule

## 6. Analytics to Track
- DAU (daily active users)
- Retention (day 1, day 7, day 30)
- Average session length
- Where players quit (which zone, which level)
- Revenue per update
- Most/least popular features
