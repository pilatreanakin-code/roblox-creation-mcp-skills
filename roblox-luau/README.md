# Roblox Luau — Professional Game Scripting Skill

Makes the AI write correct, clean, performant Luau code for Roblox via the Studio MCP.

## Features
- 15 hard rules that prevent every common AI mistake
- Client vs server separation enforced
- Server-side validation patterns for every game system
- Complete code templates for: currency, shop, inventory, pets, rebirth, zones, daily rewards, codes, combat, NPC AI, leaderboards
- DataStore with pcall, retry, auto-save, BindToClose
- Anti-exploit patterns (rate limiting, type checking, distance validation)
- MCP-specific patterns for creating scripts via run_code
- Performance best practices (debounce, cleanup, pooling)
- OOP and module organization

## Requirements
- Claude Code with Roblox Studio MCP

## Installation
Copy roblox-luau/ to ~/.claude/skills/roblox-luau/
